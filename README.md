## VaR, CVaR & Backtesting  Quantitative Risk Project

## Autor: Andrés Rendón
Estado: Proyecto Finalizado
Versión Final: VaR + CVaR + Kupiec Backtest
Lenguaje: Python (Colab)

## Objetivo
Construir un modelo institucional básico de medición de riesgo de mercado:

Value at Risk (VaR) — método histórico

Expected Shortfall (CVaR)

Validación del modelo mediante Backtesting (Kupiec POF Test)

Análisis institucional según Hull, Alexander y FRTB


## Contenido
Portafolio multi-activo: equities, bonos, FX, commodities, crypto

Descarga automática de datos (yfinance)

Retornos diarios y retorno del portafolio

Cálculo de VaR empírico

Cálculo de CVaR (Expected Shortfall)

Backtest del VaR (Kupiec)

Visualizaciones y análisis



## Conocimientos aplicados:
Métodos no paramétricos

Gestión de riesgo de cola

Validación de modelos (likelihood ratio)

Medidas coherentes de riesgo

Interpretación tipo mesa de riesgo
