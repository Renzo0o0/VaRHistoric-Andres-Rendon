# Historical VaR Portfolio

This project implements a multi-asset **Historical Value-at-Risk (VaR)** model using Python.

## Assets
- Equities: SPY, EEM  
- Bonds: IEF  
- FX: EURUSD=X, JPY=X  
- Commodities: GC=F (Gold), BZ=F (Brent Oil)  
- Crypto: BTC-USD  

## Features
- Daily historical VaR (95% & 99%)
- Data downloaded from Yahoo Finance
- Modular Python structure
- Portfolio construction with institutional weights

## Next versions
- CVaR / Expected Shortfall
- Backtesting (Kupiec test)
- Stress scenarios
- Institutional PDF report
